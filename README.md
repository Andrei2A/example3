# Калькулятор

Веб-приложение калькулятор, созданное на чистом HTML, CSS и JavaScript.

## Описание

Современный калькулятор с интуитивным интерфейсом в стиле iOS. Приложение работает полностью в браузере и не требует установки дополнительных зависимостей или серверной части.

## Возможности

### Математические операции
- Сложение (+)
- Вычитание (−)
- Умножение (×)
- Деление (÷)
- Вычисление процентов (%)
- Смена знака числа (+/−)

### Дополнительные функции
- Поддержка десятичных дробей
- Цепочка вычислений (можно выполнять операции последовательно)
- Автоматическое округление результатов
- Экспоненциальная запись для больших чисел
- Защита от деления на ноль
- Полная очистка (AC)

### Поддержка клавиатуры
| Клавиша | Действие |
|---------|----------|
| 0-9 | Ввод цифр |
| . | Десятичная точка |
| + | Сложение |
| - | Вычитание |
| * | Умножение |
| / | Деление |
| Enter или = | Вычислить результат |
| Escape | Очистить всё |
| Backspace | Удалить последний символ |

## Запуск приложения

### Способ 1: Двойной клик
Просто дважды кликните на файл `calculator.html` — он откроется в браузере по умолчанию.

### Способ 2: Через браузер
1. Откройте любой современный браузер (Chrome, Firefox, Edge, Safari)
2. Нажмите `Ctrl+O` (Windows/Linux) или `Cmd+O` (macOS)
3. Выберите файл `calculator.html`

### Способ 3: Через командную строку
```bash
# Windows
start calculator.html

# macOS
open calculator.html

# Linux
xdg-open calculator.html
```
